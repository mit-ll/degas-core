# unit_test

This directory runs the unit test for the SimpleTrackedSurveillance block. To run the SimpleTrackedSurveillance unit test, execute the following commands in the command window while in the SimpleTrackedSurveillance unit_test directory:

    >> suite = testsuite(pwd,'Name','unitTest_*');
    >> run(suite)
